# 第5章：便利な道具箱とミニゲーム

Javaには、最初から用意されている便利な道具箱（標準ライブラリ）がたくさんあります。

これらは `java.util` という場所にしまってあるので、使うときはコードの先頭に `import`（インポート）宣言を書く必要があります。

## 1. 配列より便利な「List」と「Map」

実務の開発では、配列(`[]`)よりも、以下の「コレクション」と呼ばれるクラスをよく使います。

`CollectionDemo.java` で動きを確認してみましょう。

### ArrayList (リスト)

配列の弱点（最初に個数を決めないといけない）を克服したクラスです。

あとから自由にデータを追加・削除できます。

* `add(データ)`: 追加
* `get(番号)`: 取り出し
* `size()`: 個数を数える

### HashMap (マップ)

「キー」と「値」をセットで保存するクラスです。

Web開発でよく使う「JSON」というデータ形式と似た考え方です。

* `put(キー, 値)`: 保存
* `get(キー)`: 値を取り出す

## 2. 最終課題：数当てゲーム (Game.java)

これまでの学習の総決算です。

`Scanner`（入力を受け取る機能）と `Random`（サイコロ機能）を使って、簡単なゲームを作ってみましょう。

### 使っている技術

1. **import:** `java.util.Scanner` などを読み込む。
2. **while(true):** 正解が出るまで無限に繰り返す。
3. **if - else:** 数字の大小を判定する。
4. **break:** ループを強制終了してゴールする。

### 実行方法

このプログラムは、あなたがキーボードで数字を入力しないと進みません。
1. 「Run」ボタンを押す。
2. 下のターミナルに「数字を入力してください >」と出る。
3. ターミナルをクリックして、数字を入力し Enter キーを押す。
4. 正解するまで頑張る！
