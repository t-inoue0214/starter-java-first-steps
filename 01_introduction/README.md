# 第1章：Javaに触れてみよう

Javaの学習へようこそ！
まずは難しい理屈は抜きにして、プログラムを動かしてみましょう。

## 1. プログラムを実行してみよう

左側のファイル一覧から `Hello.java` を開いてください。
画面の中に、以下のようなコードが書かれています。

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
        System.out.println("はじめてのプログラム");
    }
}
```
