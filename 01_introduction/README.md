# 第1章：Javaに触れてみよう

Javaの学習へようこそ！

まずは難しい理屈は抜きにして、プログラムを動かしてみましょう。

## 1. プログラムを実行してみよう

左側のファイル一覧から `Hello.java` を開いてください。

画面の中に、以下のようなコードが書かれています。

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
        System.out.println("はじめてのプログラム");
    }
}
```

### 実行の手順

1. `Hello.java` を開いて、少し待ちます（初回はJavaの準備に数秒かかります）。
2. `main` という文字の上に **「Run」** (または実行) というボタンが表示されたら、それをクリックしてください。

   > **Note**
   > もしボタンが表示されない場合は、画面上のどこかをクリックしたり、数秒待ってみてください。

3. 画面の下に「ターミナル」というパネルが開き、実行結果が表示されます。

    ```bash
    Hello, Java!
    はじめてのプログラム
    ```

この文字が表示されれば成功です！あなたは今、Javaプログラムを動かしました。

## 2. コードを書き換えてみよう

プログラムを少し変更して、自分の言葉を表示させてみましょう。

1. `System.out.println("...");` の `"` の中身を好きな言葉に変えてください。
    例: `System.out.println("Javaは楽しそう！");`
1. 変更したら、もう一度 「Run」 ボタンを押して結果を確認しましょう。
    ポイント `System.out.println`（システム・アウト・プリントエルエヌ）は、カッコの中身を表示して改行する命令です。

## 3. その場で実験！「JShell」を使ってみよう

Javaには、ファイルを作らなくてもその場でプログラムを試せる 「**JShell**（ジェイシェル）」 という便利なツールがあります。

計算やちょっとした確認に最適です。

### JShellの起動

- 画面下の「ターミナル」をクリックして入力できる状態にします。
- `jshell` と入力して `Enter` キーを押します。
- `jshell>` という表示が出れば準備完了です

### 実験してみよう

以下のコードをコピーして、ターミナルに貼り付け（または入力）して Enter を押してみてください。

#### 1. 計算してみる

```bash
1 + 1
```

```bash
100 * 25
```

#### 2. 文字を加工してみる

Javaにもともと用意されている機能（メソッド）を使ってみます。

```bash
"hello".toUpperCase()
```

Result: `"HELLO"`（大文字に変換されました！）

#### 3. 数学の機能を使ってみる `Math` という便利な道具箱を使ってみます

```bash
Math.max(10, 50)
```

Result: 50 （大きい方の数字を返してくれました）

#### JShellの終了

実験が終わったら、以下を入力して終了します。

```bash
/exit
```

## 第1章のまとめ

- Javaプログラムは「**Run**」ボタン一つで動かせる。
- `System.out.println` で文字を表示できる。
- `jshell` を使うと、ファイルを作らずに計算や実験ができる。
